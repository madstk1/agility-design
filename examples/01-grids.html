This is how simple grids are setup using Agility Design. While similar to Bootstrap and Foundation, the changes required to change the grid preset are more managable. This is because the entire grid has a size set on creation, which will apply to the following column elements.

To see the effects on grid presets, resize the browser window and see the how the grids react, according to their size preset.
